# prontuario-matern-link
Projeto de implementação de um prontuário médico para e empresa Matern Link para Residência com a Porto Digital
# Para iniciar o back-end abra o terminal de comandos (CMD) e digite em ordem:
- iniciar o banco de de dados em "../ProntuarioMaternLink/backend/initDB.sql
- cd backend "ProntuarioMaternLink/backend"
- npm install
- npm start
- acessar a pagina login.html
