# Matern Link - Sistema de Prontuário Eletrônico

Projeto de implementação de um prontuário médico para e empresa Matern Link para Residência com a Porto Digital

## Requisitos

- Node.js (versão 14 ou superior)
- MySQL (versão 5.7 ou superior)
- Navegador web

# Para iniciar o banco de dados faça:
- iniciar o banco de de dados em "backend/initDB.sql"

# Para iniciar o back-end abra o terminal de comandos (CMD) e digite em ordem:
- cd backend 
- npm install
- npm start
# Quanddo finalizado acessar a pagina
 [login.html](http://localhost:3000/login.html)
